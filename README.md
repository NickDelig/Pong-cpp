# 🎮 Pong

A 2D game built entirely in C++. This project was created to explore game loops, memory management, and object-oriented programming in C++.

## 🕹️ Gameplay
There are two players in this game. One is the user and the other is the CPU, which the user must score 10 points in order to win.

**Controls:**
* `Up Arrow`: UP
* `Down Arrow`: DOWN
* `Esc`: Pause/Quit

## 🛠️ Tech Stack & Libraries
* **Language:** Standard C++ (C++11/14/17)
* **Graphics/Audio Library:** [Raylib]
* **IDE:** Visual Studio Code
* **Compiler:** [GCC (MinGW)]
